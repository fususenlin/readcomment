readcomment
===========
 
#readcomment是什么?
一个社会化评论的读书网站
品阅中文网

向开源社区致敬

##rc有哪些功能？

* 方便的导入导出功能
    *  直接把一个txt文本文件拖放到当前这个页面就可以了
    *  直接下载txt文件
* 为每一段编写评论，让阅读成为一种享受
* 查看其它人对该段的评论，交朋友
* 分享自己的评论
